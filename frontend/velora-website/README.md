# Velora Website

A premium Interior Design & Furniture website built using the MERN Stack.

---

## Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- React Router DOM
- Axios
- Framer Motion
- React Hook Form
- React Query
- Swiper.js

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

Clone the project

```bash
git clone <repository-url>
```

Navigate to frontend

```bash
cd frontend/velora-website
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## Available Scripts

```bash
npm run dev
```

Starts the development server.

```bash
npm run build
```

Creates a production build.

```bash
npm run preview
```

Runs the production build locally.

```bash
npm run lint
```

Runs ESLint.
